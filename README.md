> Generates a text file of "fake" product data

### 🍩 [Homepage](https://github.com/klutchdev/klutch-product-generator)

## Install

```sh
npm i klutch-product-generator
```

## Usage

> Generates a .txt file of the output locally

```sh
npm start
```

## Run tests

```sh
npm test
```
